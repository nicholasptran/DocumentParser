# setup

## dependencies

[uv - python project manager](https://github.com/astral-sh/uv)

## environment

run this in your console:

```powershell
git clone https://github.com/nicholasptran/DocumentParser.git
cd DocumentParser
code .

# this installs python if needed and installs dependencies into local environment
uv sync

# run this to see the console app
uv run main.py
```
